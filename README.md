# Introducción a los Sistemas Operativos

## Conceptos Fundamentales de Sistemas Operativos

---

### OBJETIVOS GENERALES DEL CURSO

El objetivo principal de este curso es proporcionar los **conceptos fundamentales** de los Sistemas Operativos (SO) desde una doble perspectiva:

1.  El SO como **administrador eficiente de recursos** (CPU, memoria, E/S).
2.  El SO como **facilitador** de la interfaz y el uso al usuario.

Se buscará desarrollar **casos experimentales** utilizando Sistemas Operativos conocidos para afianzar el conocimiento.

---

### CONTENIDOS MÍNIMOS

El curso se estructura alrededor de los siguientes pilares de estudio:

* **Tipos de Sistemas Operativos:**
    * Clasificaciones y arquitecturas.
* **Procesos y Scheduling (Planificación) de Recursos:**
    * Concepto de proceso y ciclo de vida.
    * Algoritmos de planificación de la CPU (scheduling).
* **Administración de Memoria:**
    * Gestión de Caché.
    * Gestión de Memoria Principal (RAM).
    * Gestión de Memoria Externa (Paginación, Segmentación).
* **Control de E/S (Entrada/Salida):**
    * Manejo de dispositivos.
    * Buffers y *spooling*.
* **Administración de Archivos:**
    * Estructura, operaciones y sistemas de protección de archivos.
* **Introducción a Linux:**
    * Filosofía UNIX y comandos básicos.

Para la profundización de los temas, se recomienda la consulta de las siguientes obras:

| Autor | Título | Enfoque |
